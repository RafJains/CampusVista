# Database Schema

Python reads the MVP SQLite database at:

```text
python-backend/data/campus_seed.db
```

The database is generated by:

```text
python-tools/scripts/generate_seed_db.py
```

## Tables

### checkpoints

Outdoor graph nodes.

- `checkpoint_id`
- `checkpoint_name`
- `checkpoint_type`
- `x_coord`
- `y_coord`
- `latitude`
- `longitude`
- `description`
- `orientation`

### places

Searchable destinations mapped to checkpoints.

- `place_id`
- `place_name`
- `place_type`
- `checkpoint_id`
- `description`
- `keywords`

### edges

Weighted outdoor graph edges.

- `edge_id`
- `from_checkpoint_id`
- `to_checkpoint_id`
- `distance_meters`
- `is_bidirectional`
- `edge_type`

### crowd_rules

Static time-window crowd penalties.

- `crowd_rule_id`
- `checkpoint_id`
- `day_type`
- `start_time`
- `end_time`
- `crowd_level`
- `penalty_cost`
- `description`

### outdoor_panos

Panorama metadata by checkpoint.

- `pano_id`
- `checkpoint_id`
- `image_file`
- `thumbnail_file`
- `orientation`
- `description`

### recognition_refs

Recognition label metadata for future ML integration.

- `recognition_id`
- `checkpoint_id`
- `label_name`
- `model_label_index`
- `confidence_threshold`

### search_aliases

Alternate search strings for places.

- `alias_id`
- `place_id`
- `alias_text`
- `alias_type`
